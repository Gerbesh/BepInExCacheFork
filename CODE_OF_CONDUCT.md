# Кодекс поведения

Мы стремимся поддерживать дружелюбную и продуктивную среду для всех участников проекта.

## Допустимое поведение
- уважительное и конструктивное общение;
- готовность помогать и принимать обратную связь;
- обсуждение идей и решений без перехода на личности.

## Недопустимое поведение
- оскорбления, угрозы, дискриминация и травля;
- умышленный флуд, троллинг и срыв обсуждений;
- публикация личных данных без согласия.

## Модерация
Нарушения сообщайте через GitHub Issues с пометкой `[CoC]` или напрямую владельцу репозитория через его профиль. Модераторы могут удалять контент и ограничивать участие при повторных нарушениях.
